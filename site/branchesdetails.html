Branches and master: how to create them

How to create a branch:
  1. Go to command prompt
  2. Go to relevant directory (with cd)... cd coursera-test/site
  3. Check what branches we have
    • git branch -a
    • the * is the one we are in
  4. Create a new branch and swith to this branch
    • git checkout -b branchNameHere
  5. to swith to any branchname
    • git checkout branchName
  6. Commit to public directory
    • Check git status
    • git push
How to sync directly changes:
browser-sync start --server --directory --files "*"
