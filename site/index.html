How to commit:
  1. Go to command prompt
  2. Go to relevant directory (with cd)... cd coursera-test
  3. Enter git status - where we are ...if in red, there are changes
  4. Add what is new
    • git add . (for everything)
    • git add site/index.html (for specific file, in this case index file)
  5. Commit to local directory
    • Check git status
    • git commit -m "whatever comment is new"
  6. Commit to public directory
    • Check git status
    • git push
How to sync directly changes:
browser-sync start --server --directory --files "*"
